## 응답이 시트에 쌓이는 방식

- 제출 1건 = 시트 **1행** 추가. 첫 열은 자동 기록되는 **접수시각**.
- 컬럼 순서는 위 문항 순서와 동일 (총 30열 = 접수시각 + 29문항).
- **필요 컨설팅 항목**처럼 여러 개 고른 값은 한 칸에 콤마로 묶여 저장됩니다.
- 헤더(한글 제목 줄)는 첫 응답 때 자동 생성되고 고정됩니다.

## 운영 메모

- **실운영 시작 전**: 시트에 남아있는 테스트 행 2건(`테스트홍길동`, `E2E브라우저테스트`)을 삭제하세요.
- 폼 내용을 수정하려면 저장소의 `index.html`을 고쳐 다시 배포합니다.
- 응답을 **다른(기존) 시트**로 받고 싶으면 연결 대상만 바꾸면 됩니다.
- 전 구간(폼 → 웹앱 → 시트)은 실제 브라우저 제출로 동작 검증 완료.
